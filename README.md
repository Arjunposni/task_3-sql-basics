# Task 3 – SQL Basics (Filtering, Sorting & Aggregations)

## Objective
The objective of this task is to practice basic SQL concepts such as filtering, sorting, grouping, and aggregation using a real-world sales dataset.

## Dataset
Superstore Sales Dataset  
The dataset was imported into MySQL and contains order, customer, sales, and profit information.

## Tasks Performed
- Verified total number of records in the dataset
- Filtered data using the WHERE clause
- Sorted results using ORDER BY
- Used aggregate functions like SUM, AVG, and COUNT
- Grouped data using GROUP BY
- Applied HAVING to filter aggregated results
- Used BETWEEN for date range analysis
- Used LIKE for pattern matching
- Identified top 5 customers based on total sales

## Files Included
- `queries_task3.sql` – Contains all SQL queries written for the task
- `sales_summary.csv` – Exported summary of total sales by category
- `README.md` – Description of the task and work done

## Outcome
This task helped strengthen my understanding of SQL basics and improved my ability to analyze data using structured queries.
